Privacy Policy (EN)

Last updated: March 2026

Prompta Studio is an AI-assisted image analysis and prompt structuring tool. This Privacy Policy explains how Prompta Studio handles user data.

1. No account system

Prompta Studio does not require users to create an account and does not provide built-in user login, registration, or profile features.

2. No developer-operated data collection

Prompta Studio does not operate a backend server for collecting user content. The developer does not collect, store, sell, or share users’ images, prompt text, JSON results, API keys, usage records, or asset library content through a developer-controlled server.

3. Local storage

By default, app settings, API configuration, asset library data, image analysis results, JSON results, and local usage statistics are stored locally on the user’s device.

4. Third-party AI service requests initiated by the user

When a user actively chooses and configures a cloud AI model provider, and then starts an image analysis or text processing request, relevant input content may be sent directly from the user’s device to that third-party AI service provider.

  • User-provided images may be sent for image analysis.
  • User-entered text may be sent for text analysis or generation.
  • User-provided API keys may be used only to authenticate requests to the selected provider.

These requests are initiated by the user. Prompta Studio itself does not proxy these requests through a developer-operated server.

5. No advertising or analytics SDKs

Prompta Studio does not include built-in advertising SDKs, analytics SDKs, crash reporting SDKs, or user tracking SDKs.

6. User responsibility for third-party services

If you use cloud AI services through your own API credentials, your data may be processed according to the privacy policy and data retention practices of the selected third-party provider. Please review the policies of those providers before use.

7. Contact

If you have questions about this Privacy Policy, please contact: zhangxuming66@sina.com

隐私政策（中文）

最后更新：2026 年 3 月

Prompta Studio 是一款用于图片分析与提示词结构化整理的 AI 创作工具。 本隐私政策说明 Prompta Studio 如何处理用户数据。

1. 无账号系统

Prompta Studio 不要求用户注册账号，也不提供内置的登录、注册或个人资料功能。

2. 开发者不通过自有服务器收集数据

Prompta Studio 没有用于收集用户内容的开发者自建后端服务器。 开发者不会通过自己控制的服务器收集、存储、出售或共享用户的图片、提示词文本、 JSON 结果、API Key、使用记录或资产库内容。

3. 默认本地存储

应用设置、API 配置、资产库数据、图片识别结果、JSON 结果以及本地使用统计， 默认保存在用户自己的设备本地。

4. 用户主动发起的第三方 AI 服务请求

当用户主动选择并配置云端 AI 模型服务商，并手动发起图片识别或文本处理请求时， 相关输入内容可能会直接从用户设备发送到该第三方 AI 服务提供商。

  • 用户上传的图片可能会被发送用于图像分析；
  • 用户输入的文本可能会被发送用于文本分析或生成；
  • 用户填写的 API Key 仅用于向所选服务商发起鉴权请求。

这些请求均由用户主动发起。Prompta Studio 不会通过开发者自建服务器中转这些请求。

5. 不包含广告、统计或追踪 SDK

Prompta Studio 不内置广告 SDK、统计分析 SDK、崩溃上报 SDK 或用户追踪 SDK。

6. 第三方服务的责任边界

如果你使用自己配置的云端 AI 服务，那么相关数据将受对应第三方服务提供商的隐私政策 和数据保留规则约束。请在使用前自行查阅相应服务商的政策。

7. 联系方式

如对本隐私政策有疑问，请联系： zhangxuming66@sina.com
